# App Store スクリーンショット撮影ガイド

## 概要

「おやすみ前の1分」App Store申請用スクリーンショットの撮影ガイドです。
各画面の状態、キャプションコピー、撮影時の注意点をまとめています。

---

## 基本仕様

- **デバイス**: iPhone 6.7インチ (iPhone 15 Pro Max / iPhone 16 Pro Max)
- **枚数**: 最大10枚（推奨: 5〜6枚）
- **フォーマット**: PNG または JPEG、sRGB
- **解像度**: 1290 x 2796 px（6.7インチ）
- **向き**: 縦向き

### デザインフレーム

スクリーンショットにはフレーム（キャプション付きの装飾レイヤー）を被せることを推奨します。

- **背景**: グラデーション `#0F172A` → `#1E293B`
- **キャプション文字色**: `#F1F5F9`
- **キャプションフォント**: Hiragino Sans W6 または Noto Sans JP Bold、24〜28pt相当
- **端末モックアップ**: 実機フレームの中にスクリーンショットを配置

## スクリーンショット一覧（推奨6枚）

### 1. ホーム画面（メインビジュアル）

| 項目 | 内容 |
|------|------|
| **画面** | ホーム (`/`) |
| **状態** | 初回訪問（未記録状態）。ボタンは「話しはじめる」表示 |
| **キャプション** | **寝る前1分、声で気持ちを整理** |
| **ポイント** | 月のアイコン、タイトル「おやすみ前の1分」、サブタイトル、ラベンダー色のCTAボタンが全て見える状態 |
| **撮影メモ** | ダークモード、ステータスバーは時刻 21:30 前後に設定すると雰囲気が出る |

### 2. AI対話画面（録音前）

| 項目 | 内容 |
|------|------|
| **画面** | 録音 (`/recording`) |
| **状態** | AIの初回メッセージ「こんばんは。今日はどんな一日でしたか？」が表示されている。録音開始前 |
| **キャプション** | **AIがやさしく聞いてくれる** |
| **ポイント** | AIの吹き出し（半透明白）、画面下部のマイクボタン（白円）、波形ビジュアライザーが見える |
| **撮影メモ** | マイクアイコンが中央下に配置されていること |

### 3. AI対話画面（会話中）

| 項目 | 内容 |
|------|------|
| **画面** | 録音 (`/recording`) |
| **状態** | ユーザーが1〜2回発話し、AIが返答している状態。吹き出しが3〜4つ並ぶ |
| **キャプション** | **話すだけ。文字を打つ必要なし** |
| **ポイント** | ユーザー吹き出し（ラベンダー、右寄せ）とAI吹き出し（半透明白、左寄せ）の対話が見える。「話し終えた」ボタンも表示 |
| **撮影メモ** | 会話内容はデモ用テキストを事前に用意。自然な会話例を入れる |

**デモ会話例:**
```
AI:  こんばんは。今日はどんな一日でしたか？
User: 今日は仕事が忙しくて疲れたけど、帰りに友達とごはん食べて楽しかった
AI:  忙しい中でも友達との時間が持てたんですね。それは素敵な一日ですね
```

### 4. 感情サマリー画面

| 項目 | 内容 |
|------|------|
| **画面** | サマリー (`/summary`) |
| **状態** | 感情分析結果が表示されている。EmotionCardに感情ラベル・サマリー・スコアバーが見える |
| **キャプション** | **今日の気持ちをAIが見える化** |
| **ポイント** | 「今日のきもち」タイトル、感情バッジ（例: 充実 / オレンジ）、サマリーテキスト、強さバー、「また明日も話しかけてね」メッセージ |
| **撮影メモ** | ポジティブな感情ラベル（充実、喜び、平和のいずれか）で撮影するのが望ましい |

**デモデータ例:**
```
感情: 充実（#FF8C42）
スコア: 75%
サマリー: 忙しい一日の中にも、友人との温かい時間を大切にできた一日でした
```

### 5. 週間チャート画面

| 項目 | 内容 |
|------|------|
| **画面** | 週間 (`/(tabs)/weekly`) |
| **状態** | 7日分のデータが表示されている。各日に感情色の丸とバーが並ぶ |
| **キャプション** | **1週間の心のリズムを一目で** |
| **ポイント** | 「今週のきもち」タイトル、7日分のカラフルな感情ドット・バー、日付と感情ラベル |
| **撮影メモ** | 事前に7日分のダミーデータを入れて撮影。色のバリエーションが出るよう異なる感情を混ぜる |

**デモデータ例:**
```
月: 疲れ（#7B68EE）/ 火: 不安（#9B8EA8）/ 水: 平和（#87CEEB）
木: 充実（#FF8C42）/ 金: 喜び（#FFD700）/ 土: 平和（#87CEEB）/ 日: 充実（#FF8C42）
```

### 6. ホーム画面（記録済み状態）

| 項目 | 内容 |
|------|------|
| **画面** | ホーム (`/`) |
| **状態** | 今日の記録が完了している。ボタンが「今日はもう話したね」に変わり、半透明で非活性風の状態 |
| **キャプション** | **毎晩の習慣に。お疲れさま** |
| **ポイント** | 完了状態のボタン（opacity下がった状態）が「今日はもう話したね」と表示 |
| **撮影メモ** | 1回記録を完了してから撮影する |

## 撮影手順

1. **環境設定**
   - iOS シミュレーター（iPhone 15 Pro Max）またはTestFlight実機を使用
   - 時刻表示を 21:30 前後に変更（シミュレーターの場合 `xcrun simctl status_bar` で変更可能）
   - Wi-Fiアイコン、電波フル表示に設定

2. **ダミーデータの準備**
   - 週間チャート用に7日分のエントリを AsyncStorage に準備
   - 感情サマリー用に自然な会話・感情データを準備

3. **撮影順序**
   - ホーム画面（未記録） → 録音画面（初期） → 録音画面（会話中） → サマリー画面 → 週間チャート → ホーム画面（記録済み）

4. **フレーム加工**
   - Figma、Canva、またはScreenshots Creator等のツールでフレーム付き画像を作成
   - キャプション + 端末モックアップのレイアウトに加工
