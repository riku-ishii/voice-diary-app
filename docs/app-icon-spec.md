# アプリアイコン デザイン仕様書

## 概要

「おやすみ前の1分」のアプリアイコンデザイン仕様です。
夜の安心感・声による自己表現・習慣化のテーマを1つのアイコンに凝縮します。

---

## コンセプト

### キーワード
- **夜 / おやすみ** -- ダークカラーと月のモチーフ
- **声 / 対話** -- 音波や吹き出しの要素
- **安心 / やさしさ** -- ラベンダーの柔らかい色調
- **シンプル / 習慣** -- ミニマルで毎日目に入る心地よさ

### デザインの方向性
ダークな背景にラベンダーの月と音波を組み合わせた、夜の静けさと声の温かみを感じるアイコン。

---

## カラーパレット

| 要素 | カラー | 用途 |
|------|--------|------|
| 背景グラデーション（上） | `#0F172A` | アイコン背景の上部 |
| 背景グラデーション（下） | `#1E293B` | アイコン背景の下部 |
| メインモチーフ | `#7B68EE` | 月 / 音波のメインカラー（ミディアムスレートブルー） |
| ハイライト | `#A78BFA` | 月のハイライト、グロー効果 |
| アクセント | `#F1F5F9` | 音波ドットなどの白系アクセント |

---

## モチーフ構成

### プライマリ案: 三日月 + 音波

```
 構成イメージ:
 ┌──────────────┐
 │              │
 │    🌙        │   三日月（ラベンダー、左上〜中央配置）
 │   )))        │   音波（3本の弧、月から右下に広がる）
 │              │
 └──────────────┘
```

- **三日月**: アイコン中央やや左上に配置。`#7B68EE` をベースに `#A78BFA` のグロー
- **音波**: 月の右下から3本の弧が広がる。`#F1F5F9`（不透明度 40%/60%/80%）
- **背景**: `#0F172A` → `#1E293B` の縦グラデーション
- **全体の雰囲気**: 月が声を聴いている、夜に声を放つイメージ

### セカンダリ案: 月 + 吹き出し

```
 構成イメージ:
 ┌──────────────┐
 │              │
 │    🌙        │   三日月（ラベンダー、中央配置）
 │     💬       │   丸みのある吹き出し（月の隣に小さく）
 │              │
 └──────────────┘
```

- **三日月**: 中央配置。プライマリ案と同仕様
- **吹き出し**: 月の右下に小さな丸い吹き出し。`rgba(255,255,255,0.15)` の半透明
- **背景**: 同上

## サイズ仕様

App Store および iOS で必要なアイコンサイズ:

| 用途 | サイズ (px) | 備考 |
|------|-------------|------|
| App Store | 1024 x 1024 | 必須。角丸はシステムが適用するため、正方形で書き出し |
| iPhone (3x) | 180 x 180 | ホーム画面 |
| iPhone (2x) | 120 x 120 | ホーム画面 |
| iPad | 167 x 167 | iPad Pro |
| iPad | 152 x 152 | iPad |
| Spotlight (3x) | 120 x 120 | 検索結果 |
| Spotlight (2x) | 80 x 80 | 検索結果 |
| Settings (3x) | 87 x 87 | 設定画面 |
| Settings (2x) | 58 x 58 | 設定画面 |
| Notification (3x) | 60 x 60 | 通知 |
| Notification (2x) | 40 x 40 | 通知 |

### 注意事項
- 角丸マスク（iOS標準のスーパー楕円）はシステムが自動適用するため、書き出しは正方形
- 透過なし（背景は必ず塗りつぶす）
- アイコン内にアルファチャンネルは使用不可

## デザインガイドライン

### グリッドとセーフエリア
- Apple の App Icon Grid に準拠（1024x1024基準）
- 角丸マスクにより四隅約15%がカットされるため、重要な要素はセーフエリア内に収める
- セーフエリア: 中心から半径約410px以内（1024x1024基準）

### 視認性
- 小サイズ（40x40）でもモチーフが判別できること
- 月のシルエットが最も重要な識別要素
- 音波は小サイズでは省略されても月だけで成立するデザインにする

### 質感
- フラットデザインベース
- 月に軽いグロー効果（ぼかし半径 8〜12px、`#A78BFA` 40%不透明度）
- 過度なグラデーションやシャドウは避ける

## 書き出し設定

- **フォーマット**: PNG（透過なし）
- **カラースペース**: sRGB
- **解像度**: 1024 x 1024 をマスターとして各サイズに書き出し
- **Xcode**: Assets.xcassets の AppIcon セットに配置（Xcode 15以降は1024x1024の1枚のみで自動生成可能）
